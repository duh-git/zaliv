Персональные данные, указанные в п. 1.1, не являются общедоступными.

Обработка персональных данных осуществляется в следующих целях:

Основанием для обработки персональных данных является:

  • настоящее согласие на обработку персональных данных.

В ходе обработки с персональными данными будут совершены следующие действия:

  • удаление;

6.1. Персональные данные обрабатываются до отписки физического лица от рекламных и новостных рассылок. Также обработка персональных данных может быть прекращена по запросу субъекта персональных данных.
