# Как настроить уведомления

## Создание гугл таблицы и подключение App Script

1. Создать гугл таблицу, переименовать лист на "Заявки" и добавить заголовки:
    - `timestamp`
    - `phone`
    - `form`
2. Вкладка "Расширения" -> App Script
3. Авторизоваться и дать все необходимые права приложению
4. Повторить шаг 2, чтобы скрипт наверняка привязался к таблице
5. Заменть весь код в новом проекте на [broker.js](broker.js)
6. Заменть переменные в начале файла на свои данные (как получить описано ниже)
7. "Начать развертывание" -> "Новое развертывание"
8. "Выберите тип (шестеренка)" -> "Веб-приложение" ->
    - Описание: любое
    - Запуск от **моего имени**
    - Доступ у **всех**
9. Предоставить права, если будет необходимо
10. Получить **ссылку веб-приложения (URL)** и вставить в [main.js](main.js) в начале файла или **отправить разработчику**
11. Вернуться в App Script, нажать кнопку "Выполнить" и дать все необходимые права

> При изменении скрипта необходимо запускать скрипт, давать права и обновлять ссылку в main.js

## Телеграм бот

1. Создать бота с помощью [@BotFather](https://t.me/BotFather) и получить **API ключ бота**
2. Перейти по ссылке (тегу) к созданному боту и **нажать "Start"**
3. С помощью [@userinfo3bot](https://t.me/userinfo3bot) узнать **свой ID**
4. Вписать полученные **API ключ** и **свой ID** в файл App Script
